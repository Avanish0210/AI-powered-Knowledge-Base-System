How DocBrain works

A clean RAG pipeline from upload to intelligent answers.

01

Upload PDF

Upload your PDF document through the drag-and-drop interface. The file is sent to the Spring Boot backend.

02

Extract & Chunk

The PDF is parsed page-by-page, then split into 800-token chunks with metadata tags for the source document.

03

Generate Embeddings

Each chunk is converted to a vector embedding using nomic-embed-text via Ollama, then stored in PGVector.

04

Ask & Answer

Your question triggers a semantic search (top-4 chunks, 0.7 threshold), feeds context to Gemma 3, and returns an AI-generated answer.
